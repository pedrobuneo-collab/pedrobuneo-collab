## 🚀 Sobre mim
> Acadêmico em Desenvolvimento de Sistemas com sólida base em lógica de programação e arquitetura de software. Minha atuação é focada na intersecção entre a Engenharia de Dados e a Infraestrutura de TI, explorando a fundo a administração de sistemas Linux (Ubuntu, Debian, SparkLinux e Fedora) e ambientes de virtualização.

Possuo expertise técnica em Java e JavaScript, além de domínio em metodologias de padronização através de SOP (Standard Operating Procedures), garantindo a entrega de soluções escaláveis, documentadas e de alta precisão. Complemento meu perfil técnico com uma visão estratégica em Marketing Digital, unindo eficiência operacional à compreensão de métricas e resultados de negócio.
